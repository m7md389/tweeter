# Tweeter

## What is it?

**Tweeter** is a Frontend website built during the bootcamp to practice on jQuery.
It's a **mini-Twitter** simulation, where you can add/delete posts and comments.

## Stack

- HTML
- CSS
- JavaScript
- jQuery
